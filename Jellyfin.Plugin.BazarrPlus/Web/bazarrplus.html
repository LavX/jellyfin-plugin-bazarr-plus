Bazarr+ Subtitles

This plugin fetches subtitles from a self-hosted Bazarr+ instance via the OpenSubtitles-compatible REST endpoint. Enable the External Integration endpoint in your Bazarr+ Settings first.

e.g. http://bazarr.local:6767. No trailing slash. The plugin appends /api/v1/... itself.
Paste the token shown in Bazarr+ Settings → External Integration.
Bazarr+ is not affiliated with or endorsed by OpenSubtitles.com.
